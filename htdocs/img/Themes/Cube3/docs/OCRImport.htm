OCRImport
Klicka på knappen GivOCR för att skapa en pdf med en inbetalningslista.

Klicka på knappen kontrolluppgift för att skapa en pdf med kontrolluppgift för OCR importen.

Använd Ta Bort funktionen till höger för att ta bort OCR filen samt alla inbetalningar från systemet.

Förklaringar för OCR status:
Paid = inbetalningen överensstämmer med givarprojekten.
PaidOverflow = inbetalningen är större än givarprojekten. Överflödet går in på 51
UnderPaid = inbetalningen är mindre än givarprojekt.
Error =  inbetalningen kan inte kopplas till något givarprojekt (GiftInfoNotFound). Pengarna går in på 51. Eller givare kan inte hittas (GiverNotFound). Pengarna går in på 51 och kopplas till givare -90000.
