Fadderbarn
Använd de olika knapparna för att söka, kopiera, ändra, spara etc.

Klicka på Lista för att komma till listläge.

Nytt Barn
Klicka på Ny för att lägga till ett nytt barn. Fälten Id (identifikationsnummer) och Reg (datum för registrering) fylls i automatiskt. Klicka på Tilldela Barnnummer för att ge barnet ett barnnummer (aktivera barnet). Ett barn som inte har ett barnnummer (inaktivt barn) kommer inte med på barnlistor m m

Ta bort barn
Man kan aldrig ta bort ett barn däremot kan man inaktivera barnet genom att klicka på Inaktivera.

Tags
Taggar (tags) är ett sätt att kategorisera barn på olika sätt. T ex handikappade eller de som är i behov av operation.

Givarprojekt
Givarprojektrutan visar en summering av allt vad en givare har betalat in till olika projekt.

Klicka på Nytt Givarprojekt för att lägga till ett nytt givarprojekt - även om givaren redan har en summa (kr/mån) till det projektet. Systemet kommer då visa en total för alla givarprojekt som är kopplade till samma projekt. T ex två fadderbarn i Indien skulle innebära två givarprojekt á 200kr/mån till projekt 13 men visas som ett projekt på 400kr/mån till projekt 13. Klicka på Utöka för att se det uppdelat.
OBS funktionen är lite knölig i dagsläget men kommer bli bättre.

Handlingar
Denna rutan ersätter rapportdatum från Avanti. Den visar vilka datum rapporter skickat men visar också när ett barn blivit aktiverat eller inaktiverat. Klicka på aktuell handling för att lägga till mer info om t ex varför ett barn blivit inaktiverat.
