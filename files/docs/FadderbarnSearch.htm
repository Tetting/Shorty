Sökningar
Sökningar kan göras i alla fält. Till exempel skriv Emanuel i namnfältet och klicka på sök för att söka på alla som heter Emanuel i registret.

Lägg till O i Landskodfältet för att söka på alla som heter Emanuel och bor i Kenya. Klicka på knappen Rensa för att tömma sökfälten.

Använd tab-knappen för att hoppa mellan olika fält. Tryck Enter eller klicka på sök för att söka.
