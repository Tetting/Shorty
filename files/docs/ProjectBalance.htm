ProjektBalanser
Använd denna sida för att justera balanser på olika projekt genom.

1. Ange önskat projekt i fältet projekt. Klicka Go eller tryck Enter. En summering av aktuellt projekt kommer opp.

2. Ange datum.

3. Ange vad projektsaldot var det angivna datumet. Klicka på Preview eller tryck Enter.

4. Klicka Ställa balansen eller tryck Enter. Klicka på Avbryt för att avbryta.
